#  Таблица норм времени цеха 188 на 2025 год

Генератор Excel-таблицы с нормами времени и расценками на работы цеха 188.

## 📋 Описание

Этот проект создает профессионально оформленную Excel-таблицу с:
- Заголовком
- Столбцами: Наименование работ, Норма времени, Расценка, Количество
- Форматированием (границы, шрифты, выравнивание)
- Примечанием внизу
